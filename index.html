Projects

These are the projects I have done.
Python Discord Bot (April 2021 - May 2021)

Created a discord bot that would browse and share reddit posts, correctly embedding videos and images into the discord text channel. The bot was created and programmed in Python and was controlled with commands that could be issued in discord. Reddit API was used to get the posts, images, and videos to be displayed. This was created due to discord at the time not being able to embed reddit links into the text channel correctly at the time. Unfortunately this bot has become obselete due to many changes to both reddit and discord since then. Discord as of 2024 has been able to embed reddit content correctly and has for the most part made discord bots unecessary. Also, reddit has now begun charging money for access to Reddit API usage.

Library Database (Oct 2022 - Nov 2022)

Designed a functional database in Oracle intended for library use. The Database would be used by libraries to keep track of borrowed books. Created ER diagram and relational schema for the database. Created necessary and functional database tables for the database.
